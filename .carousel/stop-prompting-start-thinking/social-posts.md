## LinkedIn

Most developers open the AI first. Think later. I did that too.

The code came out technically correct but wrong. Wrong abstractions, missing conventions, no real judgment. The AI was generating, not thinking.

I spent months building a different mental model: think first, use voice instead of typing, set up context files and skills as guardrails, keep the review loop in your own hands.

The same approach works for writing, editorial work, branding. Not just code.

Swipe for the workflow. Full post in the comments.
